# How to run


## Installation

- First create & setup .env. You can use .env.example for sample
- Create new database
- Run

```bash
npm install
```

```bash
npm run migrate
```

```bash
npm run seeder
```

## Running Project

```bash
npm run dev
```

OR

```bash
npm run start
```

## Running Unit Test
```bash
npm run test
```
